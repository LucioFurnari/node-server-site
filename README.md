# node-server-site
A simple page created using node js, with index, about, contact and 404 error pages.

Tech used:
- Node Js
- Html
- Css

To start the project:
```
npm start 
```
or

````
node index.js
````

The routes are: 
```
/
/about
/contact-me

And have a 404 page for pages not listed.
```
